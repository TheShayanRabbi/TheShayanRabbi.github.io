This repository is a design investigation, not just a mockup folder.

## Core question

The README states the problem clearly: which web elements serve a genuine purpose, and which are simply "business masks" for poor design?

That question matters because many landing pages are overloaded, manipulative, and cognitively exhausting while pretending to be polished.

## Design stance

The project argues for reduction instead of decoration. A landing page should work more like a tool than a billboard.

## Current notes

The repository also captures an early visual direction through a compact color system:

- chassis blue
- signal orange
- text platinum
- indicator green
- void black
